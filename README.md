# ToDo App JS

Aplicación de gestión de tareas desarrollada con JavaScript utilizando Programación Orientada a Objetos (POO).

## Descripción

Este proyecto permite registrar usuarios, iniciar sesión y administrar tareas personales.

Fue desarrollado como práctica de Programación 1, aplicando conceptos de JavaScript moderno, manipulación del DOM y programación orientada a objetos.

## Funcionalidades

- Registro de usuarios
- Inicio de sesión
- Creación de tareas
- Cambio de estado de tareas
- Eliminación de tareas
- Navegación entre vistas
- Validaciones de datos

## Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript ES6
- Programación Orientada a Objetos (POO)
